Windows

A Windows a legelterjetebb operációs rendszer számítógépekre. Elérhető a rendszer okostelefonokra, tabletekre is, de nem piacvezető valmint sokan kritizálják.
Az operációs rendszer 138 nyelven érhető el.
A Microsoft az Apple cég operációs rendszerének alapötletét felhasználva kezdte el kifejleszteni az ablaktechnikán alapuló rendszerét, a Windows-t. A rendszer a DOS után forradalmian újnak számított, sikere pedig a konkurenciához képest alacsony árában volt keresendő.
